# vue-project

[Vue.js＋Firebaseプロジェクト作成(Authentication、Firestore、Storage、Functions)](https://qiita.com/yoshi0518/items/e16d600a06f9acb2f815)

## セットアップ
```
npm install
```

## 開発サーバー起動
```
npm run serve
```

## ビルド
```
npm run build
```

## Jest
```
npm run test
```

## ESLint、Prettier
```
npm run lint
```

## Firebaseデプロイ
```
firebase deploy

#Hostingのみ
firebase deploy --only hosting

#Functionsのみ
firebase deploy --only functions
```
